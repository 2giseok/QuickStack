#### 퀵스택 (QuickStack)
기술 블로그 AI 요약과 깃허브 릴리즈 통합 정보 제공
#### 프로젝트 소개
기술 블로그 AI 요약과 깃허브 릴리즈를 제공 합니다.
정보를 얻기 위해 기술 블로그와 릴리즈를 주로 확인을 하는데 두 가지 모두 여러 곳에 흩어져서 올라오기 때문에 찾아보기 어렵다는 점이 존재합니다. 추가적으로 기술 블로그의 경우 글의 길이가 길어서 부담스헙게 느껴질 수 있다고 판단하여, 이러한 점들을 전부 보완하여 두 가지의 정보를 전부 보여주면서  기술 블로그의 경우 AI요약정보를 제공하여 사용자에게 편리함을 제공 합니다.
